Bi-Cube Stool is a wood stool that helps people to meditate through sitting. It is not a comfortable stool to sit on. Instead, it gives mild pressure to the user to let one sit upright. The stool acts like a platform of thinking.

The Bug Bag is a 'SPIME'. It is a smart bag that mainly made of recyclable waxed hemp canvas. The bag can use NFC to recognize items marked by NFC tags. When put items inside, the e-paper screen can display the contents. It is a transparent container you can look through and can organize other objects.
